# DotNetprojet2

Pour lancer le projet, il est possible de le faire en ligne de commande ou bien directement depuis Visual Studio.

Pour ceux préférant utiliser la ligne de commande:
- Cloner le repository en local avec la commande: git clone https://github.com/S-Marnat/OC-Projet2
- Se placer dans le répertoire du projet: cd OC-Projet2/P2FixAnAppDotNetCode
- Lancer l'application avec la commande: dotnet run
- Cliquer sur le lien fournit (ex.: http://localhost:62929)
- Pour quitter l'application, fermer le navigateur et faire un ctrl + c dans le terminal.

Pour ceux préférant utiliser Visual Studio:
- Ouvrir Visual Studio et choisir l'option "Cloner un dépot"
- Rentrer l'adresse du dépôt: https://github.com/S-Marnat/OC-Projet2
- Choisir un dossier dans lequel cloner le repository
- Une fois la solution chargée, dans la barre d'outils, cliquer sur l'engrenage (l'élément de démarrage) et sélectionner "P2FixAnAppDotNetCode"
- Lancer l'application en cliquant sur la flèche verte "IIS Express"
- Pour quitter l'application, fermer le navigateur.
